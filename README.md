# Model-Template

This project contains all of the tools required to train any neural network without the specification of loss function and model architecture. 

## Installation

1. Clone Project
```
git clone https://github.com/mtgith/Model-Template.git
cd Model-Template
```

2. Setup Virtual Environment
```
conda create -n modelenv python=3.10
conda activate modelenv
```

3. Install Requirements
```
pip install -r requirments.txt
```
